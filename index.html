Classic Fortress

Team Fortress like it used to be

Classic Fortress is a complete Team Fortress solution. Every aspect of the game is controlled by the authors to make the experience authentic yet polished. By controlling the client, server and qwprogs (modification code) as well as the packaging, every detail can be tailored for the purpose of making this classic game accessible to anyone. The aim of the project is to add functionality and ease of use to the classic Team Fortress modification without affecting gameplay. Please note that this is not TFC. The original Team Fortress was a modification to Quake (1), which this project is based on.

Please see our wiki for more information.

Client-side features:

  • Customizable grenade timers
  • Customizable action menus
  • Options menu optimized for Team Fortress
  • Download Client Package

Note: Since this is an online installer, Windows might classify it as malicious. To bypass this, you need to click "More info" followed by "Run anyway" (Windows 8).

Server-side features:

  • Cleaned up code from TF 2.9
  • Grenade timers
  • Weapon slots
  • Reworked menu system
  • Demoman detpack is now a weapon
  • Support for older QuakeWorld clients
  • Download Server Package (Linux)
  • Download Server Package (Windows)

Note: Don't forget to chmod +x before running the installation script.

Classic Fortress is based on ezQuake, mvdsv and TF 2.9 and was created by hifi and Empezar.
